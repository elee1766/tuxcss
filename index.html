tux.css

classless css for monospace fonts, with a flat and dense aesthetic.

no javascript. no classes. plain html. ~3kb gzipped. vendor it and make it yours.

vendor (recommended)

curl -O https://github.com/elee1766/tuxcss/releases/latest/download/tux.css
curl -O https://github.com/elee1766/tuxcss/releases/latest/download/tux.min.css

cdn link

<link rel="stylesheet" href="https://elee1766.github.io/tuxcss/tux.min.css"
      integrity="__SRI_MIN__" crossorigin="anonymous">
<link rel="stylesheet" href="https://elee1766.github.io/tuxcss/tux.css"
      integrity="__SRI_FULL__" crossorigin="anonymous">

font (optional)

tux.css defaults to Cousine with monospace fallbacks. add this if you want Cousine from google fonts:

<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Cousine:ital,wght@0,400;0,700;1,400;1,700&display=swap">
